Телеграм бот для проверки заданий

Слои:
 - Адмиистратор
 - Преподователь
 - Студент

Администратор:
 - 1. Назначает преподователей
 - n. Может блокировать пользователей

Преподователь:
 - Закрепляет для себя учеников
 - - Для закрепления можно использовать ссылку преподователя
 - Раздает задания, на определенные группы студентов
 - Может проверять задания, используя бальную систему и/или комментарии

Студент:
 - Регистрирует институт к которому принадлежит
 - Регистрирует группу
 - Может вносить ответы на задания
